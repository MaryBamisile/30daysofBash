#### Bash structure

A basic Bash script typically follows this structure:

1. Shebang
2. Comments/Documentation
3. Global variables
4. Functions
5. Main script logic
```bash

3)

random_number=$(generate_random_number)

4)

*)

;;

esac

done

echo "Script execution completed."
```
